FAQ

Can I move my TradeZella history into TradeHelp?

Yes — export your trades from TradeZella as CSV, then use TradeHelp's Import. The column mapper handles any CSV with headers, and duplicate rows are detected and skipped if you import twice.

Is TradeHelp really offline? What about the AI?

The journal itself never needs internet. The AI coach runs on your own machine through Ollama — a free, local model runner — so coaching works with the network cable unplugged. Optional extras (price ticker, economic calendar, updates) use the network only if you leave them on. More on the offline journal page.
