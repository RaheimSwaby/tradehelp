First launch? Quick note for Windows & macOS

Windows: SmartScreen may warn because the app isn't code-signed yet — click More info → Run anyway.

macOS: It isn't notarized yet, so after dragging TradeHelp into your Applications folder, open Terminal (Applications → Utilities) and run this once, then open the app normally:
